THE PROTODECK controller features:
- 87 potentiometers
- 90 buttons
- 81 rgb leds
- 2 PIC 18F4620 (20MHz RISC processors)
- fully custom rgb led drivers
- fully custom firmware
- 2 MIDI IN/OUT interface
- power supply unit included in the box
