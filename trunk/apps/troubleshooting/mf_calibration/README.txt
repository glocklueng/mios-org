MF Calibration Test

This application allows you to calibrate the motorfaders
See also http://www.ucapps.de/mbhp_mf.html for the details

Required Hardware: 8 motorfaders, 8 buttons, 1 LCD
                   1 CORE, 1 MF, 1 DINX module

Use motorfader #1 to move the other motorfaders
Use button 1/2 to set the "PWM Duty Down" Value
Use button 3/4 to set the "PWM Duty Up" Value
Use button 5/6 to set the "PWM Period" Value
Use button 7/8 to set the "AIN Deadband" Value
Use button 9/10 to set the "MF Deadband" Value
