public class MIOSStudioSID
extends javax.swing.JApplet

Driver class for launching MIOS Studio + SID V2 application. MIOS Studio can be launched as an application or embedded in a a web page as an applet. When launched as an application, the app shows a splash screen and is launched within its own frame. Preferences are saved and restored from this class.
